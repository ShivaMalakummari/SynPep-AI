SynPep-AI  is an end-to-end AI system that generates and optimizes synthetic peptide sequences targeting the \*\*EGFR cancer protein\*\* using Transformers and Reinforcement Learning.



&nbsp; # Overview :

This project demonstrates a complete AI-driven drug discovery pipeline:

\- Peptide sequence generation using Transformer models

\- Controlled sampling for diversity

\- EGFR-inspired binding proxy scoring

\- Reinforcement learning for self-improving peptide design
